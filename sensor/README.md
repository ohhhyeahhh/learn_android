# **Sensor**
### Overview

> 大多数 Android 设备都有内置传感器，用来测量运动、屏幕方向和各种环境条件。这些传感器能够提供高度精确的原始数据，非常适合用来监测设备的三维移动或定位，监测设备周围环境的变化，并且能够广泛地应用于各类软件和游戏中。

本下属分支项目主要介绍Android传感器的种类、概念以及使用方法。
